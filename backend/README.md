# Backend Основной проект для системы расписания дежурной смены в вузе. 

## Обзор проекта

Это API приложения для управления дежурствами в вузе включает:

- Управление студентами и группами
- Система расписаний и типов дежурств
- Автоматическая генерация дежурных с учетом рейтинга дежурств
- Управление отсутствиями и отменой/переназначением дежурств
- Аутентификацию и сохранение данных при смене устройства

Логика приложения определяет дежурного, исходя из общего рейтинга дежурств.  
Таким образом, дежурит всегда тот, кто делал это наименьшее количество раз и наибольшее количество раз пропускал свое назначенное дежурство.  
Справедливость для дежурных!

## Технологии

- **Фреймворк**: NestJS (Node.js)
- **База данных**: PostgreSQL через TypeORM
- **Аутентификация**: Passport.js JWT
- **Документация + валидация**: Swagger/OpenAPI
- **Тестирование**: Jest
- **Линтинг**: ESLint + Prettier
- **Хеширование паролей**: bcrypt

├── src/

│   ├── main.ts                      # Точка входа приложения
│   ├── swagger.ts                   # Конфигурация Swagger

## Установка и запуск

### Из системы
```bash
# Установить зависимости
npm install

# Запустить в разработке (hot reload)
npm run start:dev

# Запустить в производстве
npm run start:prod
```
